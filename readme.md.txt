# Smart Wearable–Integrated IoT Air Quality Monitoring and Prediction System

## Overview

An IoT-based air quality monitoring and prediction system using ESP32, environmental sensors, and machine learning models for real-time AQI monitoring and personalized health alerts.

## Features

* Real-time air quality monitoring
* AQI prediction using XGBoost + GRU
* ThingSpeak cloud integration
* Wearable vibration alerts
* Dashboard visualization

## Technologies Used

* Python
* ESP32
* ThingSpeak
* XGBoost
* GRU
* IoT Sensors
* Machine Learning

## Sensors Used

* MQ-7
* MQ-135
* PM2.5 Dust Sensor
* DHT11

## Future Improvements

* Mobile application integration
* TinyML edge prediction
* Multi-location monitoring using ESP32 mesh
